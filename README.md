This is the source code for the personal portfolio website of **Edham Arief Dawillah**.

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3

- **Hosting**: GitHub Pages
- **Custom Domain**: [dawillah.org](https://dawillah.org)

## 📂 Project Structure
- `index.html`: Original landing page.
- `index02.html`: Updated landing page with expanded professional details.
- `css/`: Stylesheets for the site.
- `js/`: Client-side logic and external libraries.
- `images/` & `font/`: Static assets.
