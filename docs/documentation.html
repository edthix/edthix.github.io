I. Template Customization

This documentation is a step by step guide to customize and extend the XXX Template.
For this purpose you will need a code editor. I recommend to use Sublime Text 2, but you can use any other as well.
Now take about 20-30 minutes and the template is ready to go!

  1. Skin and Color Scheme

    2. Go to the demo and select your skin and color scheme.
    To apply the skin and color scheme you have to open the index.html file and edit around line 15 the "href" attribute for the second stylesheet file. 

    <link rel="stylesheet" type="text/css" href="css/d_green.css">
    Replace "d" with the skin you have chosen (d for dark and l for light) and "green" with the color you have selected.

  2. Background Picture

    3. If you want to edit the background picture behind the avatar you have to open your css file yo have linked to in the first step. Open the css file and find the declaration for .container-left-picture. This must be anywhere around line 200. Edit the url of the background property to link to your image.
    Note: If you want to create the same blur effect as you can see in the demo use the Photoshop filter "Gaussian Blur" with the radius of 3 (see the example file in the PSD folder).

  3. Avatar + Social Media

    4. To apply your own picture open the PSD folder and open the included file Avatar.psd. Delete the "avatar" layer, add your photo, mask it with the frame layer and resize it that it fits perfectly into the frame.
    Afterwards save the image as avatar.png file to the images-folder.
    To edit the links to you Social Media sites you have to open the index.html file and scroll down to around line 50 and edit the list with the class name "social-media". Place the link to the specific site in the href-attribute and choose the icon class name out of the following list:

  4. Personal Facts

    5. Now scroll down to around line 70 and edit the list with the class name "personal-info". Add or remove the list items and edit the content inside the span elements.

  5. Resume + vCard

    6. If you want to provide the posibility for your users to download your Resume or your vCard scroll down to around line 86. Edit the href attributes of the link for the first button to direct to your Resume and the link of the second button to direct to your vCard.
    If you want to display only one button remove the whole div surrounding the buttons.

  6. Twitter Widget

    7. The Twitter widget has changed. In the older versions of Elox you had to edit the Twitter user name you want to display in the custom.js file.
    Since Twitter has changed its API for fetching Tweets we have to use the Twitter widget delivered by Twitter ifself.
    Login to your Twitter account and go to the widgets section. Click on "Create new", leave all options as they are and click below on the "Create widget" button. Now as you may have noticed the URL has changed. There must be a long number in it as in the following example:

    https://twitter.com/settings/widgets/352092632815656969/edit?focus_textarea=1¬ice=WIDGET_CREATED

    Copy that number to your clipboard.
    Now open the index.html file and scroll down to around line 105. Replace the value of the data-widget-id with your copyed number and type in the Twitter user you want to follow in the data-screen-name attribute.
    Save the index.html and the widget should work fine!

  7. Title + Introduction

    8. To edit the title of the template open the index.html again and go to around line 9. Edit the title to what you think fits best for you. The same goes for the actual title which will be displayed in the browser. For this reason scroll down to around line 138. Edit the title and just below the introduction text.

  8. Education/Employment

    9. The construction for the education and employment section follows basically the same pattern. 

    <li>
  <div class="time">2012-present</div>
  <div class="station">Apple Inc.</div>
  <div class="position">Junior Art Director</div>
  <div class="position-desc">Stet clita kasd gubergren, no sea takimata 
    sanctus est Lorem ipsum dolor sit amet.</div>
</li>

    For every element you want to add to the education/employment list you have to add a new list item (<li></li>) with the div's "time", "station", "position" and optional "position-desc" inside of it. Just as the example above.
    You can find the employment list around line 153 and the education list around line 180.

  9. Skills

    10. To show you visitors your skills you have in various fields you can either use the table style or the circle style.
    For the table style layout scroll down to around line 220. Every <tr></tr> element represents a single skill you want to present.

    <tr>
  <td class="item"><div class="outer-bar">
    <div class="inner-bar" style="width:90%;">HTML</div></div></td>
</tr>

    Just edit the width and the declaration. Either remove or add a <tr></tr> element to the table to present more or less skills.

    For the circled styled skills presentation you can use the following template: 

    <li>	
  <div class="pie-container">
    <div class="pie"><div class="pie-bar"></div></div><div class="pie-inner" data-height="85">
    <span>85%</span></div>
  </div>
<div class="pie-desc">Management</div>
</li>

    You can find the circled presentation around line 240. Add or remove the list items and edit the data-height attribute, the value between the span-tags and the description.

  10. Portfolio

    11. The image above shows the basic layout structure for the portfolio. Each row contains one or two images.
    When you place two "g-2" container beside each other each container will take 50% of the whole width. Inside of those g-* containers the actual portfolio items are placed.
    You can find the portfolio starting at around line 270.
    First you have to insert the image that will be shown on the screen and then you have to link to the original image. After that you have to give that item a headline (<h4></h4>) and optional a short description. You can add a link for your visitors to the actual project so that they can take a closer look.

  11. Contact Form + Google Maps

    12. Last but not least we want to set up the Contact Form and Google Maps.
    First of all you have to go to the inc folder and open the contact_form.php file. Go to line 9 and type in your email address.
    To display the right address in Google Maps you have to go to the js folder and open the custom.js file. Go to line 5 and type in your address.
